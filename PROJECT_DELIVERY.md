## 🎯 Usage Instructions

### ⚡ Quick Start (Copy & Paste)

```bash
# Step 1: Clone & Navigate
git clone https://github.com/yourusername/Local3R.git
cd Local3R

# Step 2: Automatic Setup (Choose your OS)
# Windows users:
setup.bat

# Linux/Mac users:
bash setup.sh

# Step 3: Start MongoDB (in separate terminal)
mongod

# Step 4: Start Backend (in separate terminal)
cd backend
npm run dev

# Step 5: Start Frontend (in separate terminal)
cd frontend
npm run dev

# Step 6: Access the Application
# Open your browser:
# http://localhost:3000              (Frontend)
# http://localhost:3000/admin        (Admin Dashboard)
# http://localhost:5000/api/health   (Backend Health Check)
```

## 🧪 Testing the System

### Test the API
```bash
# Health check
curl http://localhost:5000/api/health

# Get products

# Get products by region
```
